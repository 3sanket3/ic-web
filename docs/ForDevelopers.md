# For Developers

## Prerequisites

1.  Environment: **Node.js**
1.  Tools: **yarn**

## Quick Setup

1.  Clone this repository
1.  Open terminal and run `yarn install`
1.  Then run `yarn start`
1.  Browse http://localhost:3000
